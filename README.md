# 🤖 AI Hub

AI Hub is a modern web application where users can explore and subscribe to popular AI tools.  
It simulates a SaaS (Software as a Service) platform with a clean and user-friendly interface.

---

## 🚀 Features

- 📌 Browse AI tools
- 📄 View tool details
- 💳 Monthly subscription system (UI-based)
- ⚡ Fast and responsive UI
- 🔄 Dynamic data loading using JSON

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- JSON (for data)
